# Atlas Distribuidora Analytics

## Visão do Projeto

---

## Sobre a empresa

A Atlas Distribuidora é uma empresa fictícia especializada na distribuição de materiais para construção civil.

Seu modelo de negócio é baseado no atendimento de lojas de materiais de construção, home centers, depósitos e construtoras de pequeno e médio porte, atuando em diversos estados da região Nordeste.

A empresa comercializa milhares de produtos distribuídos entre categorias como:

- Tintas
- Pisos
- Revestimentos
- Hidráulica
- Elétrica
- Ferragens
- Ferramentas
- Acabamentos

Para atender sua operação, a Atlas possui uma equipe comercial composta por representantes, supervisores e gestores responsáveis pelo relacionamento com clientes e pelo desenvolvimento de mercado.

---

## Contexto

Nos últimos anos a empresa apresentou crescimento acelerado.

Com isso aumentou significativamente o volume de:

- pedidos;
- clientes;
- produtos;
- fornecedores;
- representantes comerciais;
- filiais.

Entretanto, a tomada de decisão continua baseada em planilhas isoladas e relatórios manuais.

Essa situação gera atrasos na obtenção das informações e reduz a capacidade de resposta da gestão comercial.

---

## Problema

Os gestores possuem dificuldades para responder rapidamente perguntas essenciais para o negócio.

Os dados existem, porém encontram-se distribuídos em diferentes fontes e sem padronização.

---

## Objetivo

Desenvolver uma solução completa de Business Intelligence capaz de centralizar, organizar e transformar dados operacionais em informações estratégicas para apoio à tomada de decisão.

A solução abrangerá todas as etapas do ciclo de dados:

- Banco de Dados
- SQL
- Engenharia de Dados
- Power BI
- Python
- Documentação

---

## Escopo

O projeto contempla:

- Modelagem de Banco de Dados
- Geração de Dados Fictícios
- Consultas SQL
- Dashboard Executivo
- Análise Exploratória em Python
- Publicação no GitHub

---

Este projeto foi desenvolvido exclusivamente para fins educacionais e demonstração técnica.
